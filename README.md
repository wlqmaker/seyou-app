# SeeYou 情侣约会 App

一个情侣约会与互动的轻量 Web 应用，零依赖 Node 后端 + 单文件前端。

## 运行方法

1. 安装 [Node.js](https://nodejs.org/)（已安装可跳过，需 v14+）。
2. 在本文件夹打开终端（命令行），运行：

   ```bash
   node server.js
   ```

3. 浏览器打开 **http://localhost:3000**。

首次启动会自动创建 `db.json` 数据库文件，无需任何配置。

## 功能一览

- **双人绑定**：一方创建关系获得 6 位码，另一方用码加入，数据云端实时同步（每 4 秒轮询）。
- **首页**：毛玻璃半透明风格，背景图可自定义上传。
- **约会计划**：制定计划并生成待办清单，可「发送给 TA」，对方能修改/确认。
- **俏皮状态**：想你 / 锤你 / 揪你耳朵 / 在呢 / 忙 ing / 睡着了。
- **纪念卡 & 清单卡**：约会结束后按上传照片生成可分享纪念卡；计划可生成竖版清单卡，支持保存到本地或系统分享。
- **情侣 100 件小事**：灵感库，按 8 大分类（日常/浪漫/美食/户外/宅家/创意/旅行/深夜）筛选，一键加入计划或「想做清单」。

## 文件说明

| 文件 | 作用 |
|------|------|
| `server.js` | 零依赖 Node 后端，提供静态托管与关系/计划/照片同步接口 |
| `seeyou.html` | 全部前端（HTML+CSS+JS 内嵌），移动优先 |
| `db.json` | 运行时自动生成的本地数据库（首次运行创建，可删） |

## 备注

- 后端使用 `relationId` 作为访问凭证，请勿外泄链接。
- 照片以 base64 存入 `db.json`，长期使用建议迁移到云存储。
- 单机演示可用浏览器「无痕窗口」模拟另一半；真双人需在同一局域网或部署到公网。
